# 🔐 Password Analyzer Tool

A web-based application that evaluates password strength based on multiple security parameters.

## 🚀 Features
- Checks password length
- Validates uppercase, numbers, and special characters
- Provides strength level (Weak, Medium, Strong)
- Helps improve cybersecurity awareness

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Enter your password
2. Click "Check Strength"
3. Get instant feedback

## 🎯 Purpose
To promote safe digital practices and help users create strong passwords.
